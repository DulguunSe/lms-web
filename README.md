# Codex Сургалтын Вэбсайт

Энэ төсөл нь React (CDN), HTML, CSS ашигласан нэг хуудас бүхий сургалтын танилцуулга сайт юм. Код нь ямар ч нэмэлт хөрвүүлэлт шаардахгүй тул шууд статик файлуудаар ажиллана.

## Хэрхэн ажиллуулах вэ?

### 1. Файлыг шууд нээх
1. `index.html` файлыг хоёр даралт (double click) хийж хөтөчөөрөө нээнэ.
2. Хэрэв модал, навигаци зэрэг функцууд ажиллахгүй байвал таны хөтөчид JavaScript идэвхтэй эсэхийг шалгана уу.

### 2. Орон нутгийн серверээр ажиллуулах (сонголттой)
Энгийн статик сервер ажиллуулах нь Firebase эсвэл AJAX дуудлагуудтай уялдах үед тохиромжтой.

#### Python 3 ашиглах
```bash
python3 -m http.server 3000
```
Дараа нь хөтөч дээрээ [http://localhost:3000](http://localhost:3000) хаягаар орно.

#### Node.js ашиглах
```bash
npx serve .
```
эсвэл
```bash
npx http-server .
```

### Firebase-тэй холбох
`app.js` дотор `firebaseConfig` хэсгийг өөрийн төсөлд тохируулан оруулсны дараа Firebase Authentication болон Firestore үйлчилгээтэй холбогдоно. Хувийн түлхүүрээ нийтийн репод бүү байрлуулна уу.

## Төслийн бүтэц
```
/ (workspace root)
├── index.html      # Сайтын үндсэн HTML
├── style.css       # Загвар, өнгө, хариуцсан байрлал
├── app.js          # React бүрэлдэхүүнүүд ба UI логик
└── README.md       # Төслийг ажиллуулах заавар (энэ файл)
```
